# What is NIC?

NIC stands for Network Interface Card, It is the hardware device without which we can not connect computer to the network/internet.

## Types
1. Internal network card
2. External network card

## 1. Internal network card
- In this card, the motherboard has a slot for the network card where it can be inserted. It requires a network cable (RJ45) to provide network access.

## 2. External network card
- In desktop & laptop that doesn't havee an internal NIC, eternal NIC are used.

	- It is of two types

	1. Wireless
	2. USB cable (Wired)


## Advantages of NIC

1. Allow computer to connect without another device
2. Provides both wired and wireless facilities to connect with network
3. NIC provides also connect using IP addresses and the LAN (local area network)
4. It has a unique MAC address. 

## Disadvantages of NIC

1. Data security issues
2. configuration is not easy
3. Not portale (wireless case)
