# What is data communication?

Data communication is the process of sending or receiving the digital data betwee two or more computers via transmission medium such as a wire cable.

## Components of data communication -

- Data communication system has five components

1. **Message** - The actual data or inforamation that is to be sent.
2. **Sender** - The person who sends the message.
3. **Receiver** - One who receives the message of the sender.
4. **Transmission medium** - Actual connection establishment like cables (fibre optics cable). 
5. **Protocol** - Rules and regulation used while data transmission.
