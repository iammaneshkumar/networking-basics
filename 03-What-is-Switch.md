# What is switch?

A switch is a networking device that connects the mutiple computers together in the network.
It is mainly used to send the private messages as well as their is no wasting of data.

Switch can easily identify that which device is connected with which port by using MAC address, thats why it delivered message on the particular destinatin machine.

*Note - * Switch is more intelligent than HUB

## Advantages of switch

1. It generally used to unicast the message.
2. It provides more security than HUB.
3. Switch support full duplex data transmission mode.
4. It is used to send the data pakets based on MAC address.
5. If a node fails thier will be no effect on entire network.

## Disadvantages of switch

1. If switch fails then entire network will be failed.
2. It is more expensive.
3. It is difficult to setup.
