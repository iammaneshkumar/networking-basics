# What is HUB

1. HUB is a network device that is used to connect multiple computers in a network.
2. All the information send to the HUB is automatically send to the each port to every device
3. A HUB is less expensive, less intelligence & less complicated.
4. HUB generally used to connect computer in a LAN.
5. Transmission mode of HUB is half duplex

## Advantages of using HUB
- HUB can broadcast the message
- Less expensive
- easy installation
- Robust

## Disadvantages
- If HUB fails, then entire network fails
- Can't send private informarion/data
- Doesn't provide any security
- HUB cannot support full duplex transmission mode.
