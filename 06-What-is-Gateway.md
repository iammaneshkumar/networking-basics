# What is Gateway?

Gateway is a hardware device that is used to connect the two different and same types of networks.

It allow us to send & receive data through the internet even it is LAN network.

**Note -** It operates all 7 layers of OSI model.

## Advantages of Gateway

1. It connects two network which has different protocols.
2. It operate all 7 layers of OSI model.
3. We can't access the internet without gateway.
4. It provides some security.

## Disadvantages of Gateway

1. It is more expensive.
2. Data transmission rate is slower.
3. Difficult to maintain as well as very complex to handle.
4. It is less intelligent.
