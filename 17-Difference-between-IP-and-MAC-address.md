# Difference between IP and MAC address

| IP | MAC |
|----|-----|
| IP stands for Internet Protocol | MAC stands for Media Access Control |
| IP address is a logical address | MAC address is a physical address |
| The length of IP is 32 bit for IPv4 and 128 for IPv6 | The length of MAC address is 48 bit |
| ISP provides IP address | NIC manifacturer provides the MAC address |
| We can change the IP address of any device at any time | MAC can be changed only if we change the LAN card of the device |
| An IP address identifies a connection to a device | MAC address identifies a device |
