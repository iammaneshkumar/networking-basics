# What is Bridge?

Bridge is a network device that is used to seperate LAN into number of section.

**Note -** It operates both physical as well as data link layer of OSI model.

## Advanages of using Bridge

1. By using bridge, we can extend network.
2. It broadcast the data to each node like HUB & Repeater.
3. Collision can be reduced easily.
4. It is more intelligent.

## Disadvanatges of using Bridge

1. It does not establish connection between two different networks.
2. Once it broadcast the message, then it is incapable to stop the message.
3. It is more expensive.
4. The transmission rate of data is slower than repeater.
